## Game Modes
### Tutor
  Show a number
  Show the Binary Representation
  Let the user input that Representation
  Correct them if they're wrong
### Trainer
  Show a number
  Let the user input the correct binary
  Offer corrections
### Test
  Show a number
  Let the user input the correct binary
  Track Correct/Incorrect Answers
  Optional Timer
### Reverse Mode Trainer
  Show a Binary Representation
  Let the user input the number
  Offer Corrections
### Reverse Mode Test
  Show a Binary Representation
  Let the user inpur the number
  Track Correct/Incorrect Answers
  Optionsal Timer
### Bases
  Offer the above for Decimal, Octal, Hexadecimal, and BCD numbers
